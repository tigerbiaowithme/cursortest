# 如何运行网站

## 前提条件
✅ MongoDB Atlas 已配置
✅ .env 文件已创建
✅ 管理员账户已初始化

## 运行步骤

### 1. 打开终端
- 在项目文件夹中，右键点击空白处
- 选择"在此处打开 PowerShell 窗口" 或 "在终端中打开"

### 2. 启动开发服务器
在终端中输入以下命令：
```bash
npm run dev
```

### 3. 等待服务器启动
等待看到类似这样的消息：
```
▲ Next.js 14.x.x
- Local:        http://localhost:3000

✓ Ready in Xs
```

**重要：** 必须看到 "Ready" 消息才表示服务器启动成功！

### 4. 访问网站

#### 前端页面：
- 首页：http://localhost:3000/en
- 产品页：http://localhost:3000/en/products
- 关于我们：http://localhost:3000/en/about
- 新闻页：http://localhost:3000/en/news
- 联系我们：http://localhost:3000/en/contact

#### 后台管理：
- 登录页面：http://localhost:3000/admin/login
- 登录信息：
  - 用户名：`admin`
  - 密码：`admin123`

### 5. 停止服务器
在终端中按 `Ctrl + C` 停止服务器

## 常见问题

### 问题1：端口被占用
如果看到端口被占用的错误，可以：
1. 停止其他 Node 进程
2. 或者修改端口（在 package.json 中修改 dev 脚本）

### 问题2：页面加载很慢
- 首次访问会慢（Next.js 需要编译页面）
- 第二次访问会快很多
- 这是正常现象

### 问题3：404 错误
- 确保服务器显示 "Ready"
- 硬刷新浏览器（Ctrl + Shift + R）
- 检查 URL 是否正确

### 问题4：后台一直加载
- 打开浏览器开发者工具（F12）
- 查看 Console 标签中的错误信息
- 查看 Network 标签中的请求状态

## 快速命令参考

```bash
# 启动开发服务器
npm run dev

# 停止服务器（在运行服务器的终端中按）
Ctrl + C

# 清理缓存（如果遇到问题）
Remove-Item -Recurse -Force .next
npm run dev
```
